# /dspwn

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/dspwn <spawntype>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggles output of the spawn type despawning.
<!--cmd-desc-end-->
Accepted types: all, pc, npc, mount, pet, merc, flyer, campfire, banner, aura, object, untargetable, chest, trap, timer, trigger, corpse, item, unknown
