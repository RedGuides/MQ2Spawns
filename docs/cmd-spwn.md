# /spwn

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/spwn <spawntype>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggles output of this spawn type spawning. Accepted types:
all - pc - npc - mount - pet - merc - flyer - campfire - banner - aura - object - untargetable - chest - trap - timer - trigger - corpse - item - unknown
<!--cmd-desc-end-->
